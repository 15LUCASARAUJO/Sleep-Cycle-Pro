## About

The **Sleep Cycle** app was developed to indicate sleep and wake times, based on the science of the sleep cycle, so that the user can wake up feeling refreshed. It also contains reports on sleep and its characteristics.
>**Why this project?**  
>Waking up feeling refreshed is a problem for many people, and this is greatly influenced by sleep, both in terms of duration and quality. The sleep cycle helps determine the best times to base a night's sleep on.

---

## Features

- [ ] **Choosing a bedtime**
- [ ] **Discover the best times to wake up based on your current sleep schedule**
- [ ] **Reports on the user's sleep**

## Sobre

O **Ciclo do Sono** foi desenvolvido com a finalidade de indicar horários para dormir e acoradar, se baseando na ciência do ciclo do sono, para que o usuário possa acordar com disposição, também contém relatórios sobre o sono e suas características.
>**Por que este projeto?**  
>Acordar disposto é um problema para muitas pessoas, e isso tem muito influência do sono, tanto o tempo quanto a qualidade, com o ciclo do sono é calculado os melhores horários para basear a noite de sono.

---

## Funcionalidades

- [ ] **Escolher horário para dormir**
- [ ] **Descobrir as melhores horas para acordar baseado na hora atual do sono**
- [ ] **Relatórios sobre o sono do usuário**
